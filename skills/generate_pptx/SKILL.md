# PPT 生成 skill

## 流程

1. **母版解析**：扫描公司 PPT 模板（.pptx），提取各版式（layout）的占位符清单，作为大纲生成的"版式菜单"
2. **大纲生成**：LLM 根据版式菜单和用户事项，按固定 JSON 结构输出每页版式、标题、要点、图表数据及备注栏讲稿
3. **渲染**：python-pptx 按版式填充内容，图表使用 PPT 原生图表（可二次编辑）
4. **自检**：成品逐页转缩略图交多模态 LLM 检查溢出与排版问题，自动回修

## 输入

- `brief`：用户提供的事项说明 / 演示主题
- `template_path`：公司 PPT 模板路径（.pptx），可选
- `department`：由上层服务加载上下文

## 输出

PowerPoint 文件（.pptx），包含标题页、内容页、图表页、结尾页等，备注栏含讲稿。

## 已知约束（v1）

- 母版解析依赖模板中的命名版式和占位符；若模板无版式信息，回退到默认版式
- 图表仅支持柱状图、折线图、饼图、条形图四种原生类型
- 自检依赖多模态 LLM，若不可用则跳过自检步骤
- 模型不得编造用户未提供的数据、时间、数字
