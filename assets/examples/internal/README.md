# internal examples

Company examples are intentionally empty for now.

Add documents only after confirming:

- The document is safe to reuse internally.
- Sensitive names, prices, contracts, customer data, and personal data are removed or approved.
- The document is tagged with department and document_type.

- admin / notice: 3 good company notices
- admin / meeting_minutes: 3 good meeting minutes
- marketing / customer_solution: 3 customer-facing proposals
- marketing / product_pitch: 3 product PPTs
- product / release_note: 3 release notes
- product / prd: 3 PRDs or feature specs
